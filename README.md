# RPG Game

这是一个使用 Vue 3 + Vite 构建的 RPG 游戏项目。

## 推荐的 IDE 设置

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## 项目设置

```sh
npm install
```

### 开发模式运行

```sh
npm run dev
```

### 生产构建

```sh
npm run build
```

### 预览生产构建

```sh
npm run preview
```
